Our Projects

Projects

humess – PRS Calculation Software

We developed a scalable polygenic risk score (PRS) software solution for humess, enabling the calculation of genetic risk profiles based on GWAS data. The platform features a user-friendly web interface, a containerized backend using Nextflow and Docker, and interactive dashboards for researchers and clinicians.

Reproductive Medicine with Leogene

In collaboration with the Leogene team, we developed tools for integrating genomic data into reproductive medicine. This included genome sequencing workflows, machine learning models for embryo selection, and personalized treatment planning for improved patient outcomes.
